# sy_SecToolBox 安全工具箱开发规划文档

## 项目简介

sy_SecToolBox 是一个基于 Python + Tkinter 开发的 Windows 可视化安全工具集合，旨在为个人学习网络安全提供实用工具。项目采用轻量化设计，专注于本地和授权测试环境的安全检测与分析，严格遵循合法合规原则，仅用于技术学习研究目的。

### 技术栈
- **开发语言**：Python 3.8+
- **GUI框架**：Tkinter（内置库，无需额外安装）
- **打包工具**：PyInstaller（将项目打包为独立可执行文件）
- **网络库**：socket（内置库，用于网络相关功能）
- **其他依赖**：根据具体功能可能需要添加

## 12期详细功能介绍

### 一期：多线程端口扫描器、IP归属地查询

**1. 多线程端口扫描器**
- 支持单个IP、IP范围、域名扫描
- 支持TCP全连接扫描和SYN半连接扫描
- 可自定义端口范围和扫描线程数
- 实时显示扫描进度和开放端口
- 支持导出扫描结果

**2. IP归属地查询**
- 支持单个IP地址查询
- 支持批量IP地址查询
- 显示IP所属国家、地区、运营商等信息
- 支持复制查询结果
- 支持导出查询结果

### 二期：密码强度检测、随机安全密码生成器

**1. 密码强度检测**
- 分析密码长度、复杂度、特殊字符等
- 提供密码强度评分（弱、中、强）
- 给出密码改进建议
- 支持批量密码检测
- 支持导出检测结果

**2. 随机安全密码生成器**
- 可自定义密码长度
- 可选择包含的字符类型（大小写字母、数字、特殊字符）
- 生成多个密码选项
- 支持复制生成的密码
- 支持导出生成的密码

### 三期：文件Hash校验工具、时间戳/日期互转

**1. 文件Hash校验工具**
- 支持多种哈希算法（MD5, SHA1, SHA256, SHA512）
- 支持单个文件和批量文件哈希计算
- 支持哈希值比对功能
- 支持导出哈希结果

**2. 时间戳/日期互转**
- 支持Unix时间戳与日期时间互转
- 支持多种时间格式
- 支持批量转换
- 支持当前时间快速转换
- 支持导出转换结果

### 四期：Base64加解密、URL编解码工具

**1. Base64加解密**
- 支持文本Base64编码和解码
- 支持文件Base64编码和解码
- 支持批量处理
- 支持导出处理结果

**2. URL编解码工具**
- 支持URL编码和解码
- 支持多次编码/解码
- 支持批量处理
- 支持导出处理结果

### 五期：DNS解析查询、局域网存活主机探测

**1. DNS解析查询**
- 支持A记录、MX记录、NS记录等多种记录类型查询
- 支持批量域名查询
- 显示详细解析结果
- 支持导出查询结果

**2. 局域网存活主机探测**
- 支持基于ARP和ICMP的主机探测
- 显示存活主机IP和MAC地址
- 可自定义扫描范围
- 支持导出探测结果

### 六期：子域名简易枚举、简易HTTP模拟请求工具

**1. 子域名简易枚举**
- 支持基于字典的子域名枚举
- 可自定义字典文件
- 显示发现的子域名和IP地址
- 支持导出枚举结果

**2. 简易HTTP模拟请求工具**
- 支持GET、POST、PUT、DELETE等HTTP方法
- 支持自定义请求头和请求体
- 显示响应状态码、响应头和响应体
- 支持保存请求配置
- 支持导出请求结果

### 七期：本地端口进程监控、网络延迟测速工具

**1. 本地端口进程监控**
- 显示本地开放的端口和对应的进程信息
- 支持按端口号或进程名筛选
- 显示进程详细信息
- 支持导出监控结果

**2. 网络延迟测速工具**
- 支持ping测试和traceroute测试
- 显示网络延迟和丢包率
- 支持多目标同时测试
- 支持导出测试结果

### 八期：批量文本加密解密、特殊字符过滤检测

**1. 批量文本加密解密**
- 支持多种加密算法（AES, DES等）

- 支持批量处理
- 支持导出处理结果

**2. 特殊字符过滤检测**

### 九期：ARP缓存查看、简易日志关键词分析

- 支持刷新和清空ARP缓存

## 模块预留设计

### 工具模块预留
- **tools/** 目录下为每个工具创建独立的模块文件
- 每个工具模块包含核心功能实现
- 工具模块应提供统一的接口，便于主程序调用

### GUI模块预留
- **gui/tool_windows.py** 为每个工具创建对应的窗口类
- 工具窗口类应继承自基础窗口类，保持一致的界面风格
- 预留主题切换的接口和实现

### 工具函数模块预留
- **utils/common.py** 提供通用工具函数
- **utils/network.py** 提供网络相关的通用函数
- 预留扩展其他工具函数的空间

### 数据模块预留
- **data/** 目录用于存储必要的数据集
- 预留CVE漏洞数据库的更新机制
- 预留其他数据文件的存储位置
